# 🌐 To-Do App (Web Version)

A clean and responsive **To-Do List web application** built to help users organize their daily tasks efficiently.

🔗 **Live Demo:** https://naman-dua.github.io/TODOAPP2/

---

## ✨ Features

* ➕ Add new tasks instantly
* ❌ Delete tasks easily
* ✅ Mark tasks as completed
* 🔄 Dynamic UI updates (no page reload)
* 🎯 Minimal and user-friendly design

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

---

## 🎯 Purpose

This project was built to:

* Practice DOM manipulation
* Improve frontend development skills
* Understand user interaction in web apps
